xmlproperty 
xml:XML

Define a treeModel by XML. e.g., use XML to define one tree node, and you can get the node object by using method getNodeByAttribute. node's name is saved in node object's "nodeName" attribute, and node's value is saved in node object's "nodeValue" attribue.
