Classpublic class BitmapPool
InheritanceBitmapPool DataModel flash.utils.Proxy

Since : old version

This class is used to keep images which are loaded from outside of .swf file. All the images loaded will be changed to bitmap. This class running in singleton mode.

addBitmap()method
public function addBitmap(bitmapName:String, bitmapURL:String):BitmapLoader

Add a new image file to load Image file must be .gif, .jpeg, .jpg, .swf. If use swf or gif as image file, only the first frame can be changed to bitmap.

Parameters

bitmapName:String — Name of bitmap.(not the image file)
 
bitmapURL:String — Image file URL.

Returns
BitmapLoader

Throws
DataError — Throw NAME_EXIST error when the bitmap name already exist.
