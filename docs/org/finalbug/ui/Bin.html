Packageorg.finalbug.ui
Classpublic class Bin
InheritanceBin flash.display.Sprite
Subclasses Glaze, Icon, UIObject

Since : 2010.08

Bin is the basic class for all other display object used in fb-aslib. This class will override displayObjec's width and height. Inside class/subclasses, use displayWidth and displayHeight as object's width and height values.
